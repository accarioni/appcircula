# AppCircula: Plataforma para Cooperativas de Reciclagem

## Sobre o projeto

O AppCircula é um protótipo de aplicação web que simula a experiência de um aplicativo mobile voltado à gestão e otimização da coleta de resíduos recicláveis por cooperativas.

O projeto foi desenvolvido como demonstração prática de uma proposta de solução digital, com foco na validação de um modelo de negócios aplicado ao contexto de cooperativas de reciclagem.

Acesso à demonstração:
https://accarioni.github.io/appcircula/

---

## Objetivo

O projeto tem como objetivo explorar a aplicação de tecnologias digitais para:

* otimizar a logística de coleta de resíduos recicláveis;
* estruturar a comunicação entre geradores e cooperativas;
* aumentar a eficiência operacional das cooperativas;
* gerar valor econômico e ambiental no contexto da economia circular.

---

## Contexto acadêmico

Este projeto foi desenvolvido como atividade prática da disciplina Validação de Modelo de Negócios, no curso de Administração de Empresas da Universidade do Estado de Santa Catarina (UDESC).

A proposta envolveu a estruturação e prototipação de uma solução com potencial de aplicação prática, considerando elementos como proposição de valor, modelo de monetização, definição de público-alvo e viabilidade operacional.

---

## Tecnologias utilizadas

O projeto foi implementado utilizando tecnologias web para simular uma interface mobile:

* HTML5
* CSS3
* JavaScript

---

## Descrição da solução

A aplicação simula funcionalidades relacionadas à operação de cooperativas de reciclagem, incluindo:

* agendamento de coletas;
* visualização de rotas;
* interface de interação entre cooperativas e geradores de resíduos;
* organização de fluxos operacionais básicos.

O foco do projeto está na validação conceitual da solução e na simulação da experiência de uso, não se tratando de um sistema pronto para uso em ambiente de produção.

---

## Status do projeto

Protótipo funcional desenvolvido para fins de demonstração e validação de modelo de negócios.

---

## Considerações finais

O AppCircula deve ser interpretado como um artefato de validação voltado à análise da viabilidade de soluções digitais aplicadas à gestão de resíduos e à economia circular.
